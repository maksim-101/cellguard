    **1. Create CellGuard/Services/LocationService.swift:**

    Create an `@Observable final class LocationService: NSObject, CLLocationManagerDelegate` with:

    Properties:
    - `private let locationManager = CLLocationManager()`
    - `private var serviceSession: CLServiceSession?` (retained for iOS 18+ background delivery, BKG-02)
    - `private(set) var authorizationStatus: CLAuthorizationStatus = .notDetermined`
    - `private let monitor: ConnectivityMonitor` (injected, for updateLocation and runSingleProbe)
    - `private let eventStore: EventStore` (injected, for gap event logging)

    Initializer: `init(monitor: ConnectivityMonitor, eventStore: EventStore)` -- calls `super.init()`, sets `locationManager.delegate = self`.

    Public methods:
    - `func startMonitoring()` -- creates CLServiceSession(authorization: .always), calls `locationManager.requestAlwaysAuthorization()`, calls `locationManager.startMonitoringSignificantLocationChanges()`, sets UserDefaults `monitoringEnabled = true`, sets `lastActiveTimestamp` to current Date().timeIntervalSince1970. Mark with `@MainActor`.
    - `func stopMonitoring()` -- calls `locationManager.stopMonitoringSignificantLocationChanges()`, nils `serviceSession`, sets UserDefaults `monitoringEnabled = false`. Mark with `@MainActor`.

    CLLocationManagerDelegate methods:
    - `func locationManager(_ manager: CLLocationManager, didUpdateLocations locations: [CLLocation])` -- guard let `location = locations.last`. Call `monitor.updateLocation(latitude:longitude:accuracy:)` with the location. Call `detectAndLogGap()`. Then `Task { await monitor.runSingleProbe() }`. Finally update UserDefaults `lastActiveTimestamp` to Date().timeIntervalSince1970. Dispatch to MainActor.
    - `func locationManagerDidChangeAuthorization(_ manager: CLLocationManager)` -- update `authorizationStatus = manager.authorizationStatus`.

    Private gap detection method `detectAndLogGap()`:
    - Read `lastActiveTimestamp` from UserDefaults (Double).
    - If 0 (first launch), just set current timestamp and return.
    - Calculate gap = now.timeIntervalSince1970 - lastActive.
    - If gap > 600 (10 minutes threshold), create a ConnectivityEvent with:
      - `eventType: .monitoringGap`
      - `pathStatus: .unsatisfied` (unknown during gap)
      - `interfaceType: .unknown`
      - `dropDurationSeconds: gap` (reuse this field for gap duration)
      - `timestamp: Date(timeIntervalSince1970: lastActive)` (gap START time, not current time)
    - Insert via `Task { try? await eventStore.insertEvent(gapEvent) }`
    - Always update `lastActiveTimestamp` to now at end.

    Import: CoreLocation, Observation, Foundation.

    **2. Create CellGuard/App/AppDelegate.swift:**

    ```swift
    import UIKit
    import BackgroundTasks

    class AppDelegate: NSObject, UIApplicationDelegate {
        func application(
            _ application: UIApplication,
            didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]? = nil
        ) -> Bool {
            // Register BGTaskScheduler for supplementary background refresh (BKG-03)
            BGTaskScheduler.shared.register(
                forTaskWithIdentifier: "com.cellguard.refresh",
                using: nil
            ) { task in
                guard let refreshTask = task as? BGAppRefreshTask else { return }
                // Handled by CellGuardApp -- post notification for the app to pick up
                NotificationCenter.default.post(
                    name: .init("com.cellguard.handleRefresh"),
                    object: refreshTask
                )
            }

            // Detect location-based relaunch (DAT-03)
            if launchOptions?[.location] != nil {
                UserDefaults.standard.set(true, forKey: "launchedForLocation")
            }

            return true
        }
    }
    ```

    Note: The BGAppRefreshTask handler posts a notification because the AppDelegate does not own the ConnectivityMonitor or LocationService. Plan 03 wires up CellGuardApp to observe this notification and run the probe + gap detection.
